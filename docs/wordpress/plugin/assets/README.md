# WordPress Plugin Assets

`assets/sdk/` is generated by the root build scripts.

Public source for the generated SDK bundle lives in:

- https://github.com/gautamp8/bello-sdk/tree/main/src
- https://github.com/gautamp8/bello-sdk/tree/main/docs/wordpress/plugin
- https://github.com/gautamp8/bello-sdk/tree/main/docs/wordpress

Use:

```bash
pnpm wordpress:build
```

That writes the current `bello-embed.iife.js` bundle into the plugin so the WordPress package ships with the same runtime as the CDN/npm integrations.
