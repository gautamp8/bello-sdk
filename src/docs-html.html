Installation checklist

  1. Copy the HTML snippet for the embed script.
  2. Paste it before the closing `</body>` tag.
  3. Reload the page and click the Bello launcher.

Important attributes

These data attributes map directly to the JavaScript options when you call `Bello.push(['init', ...])`.

data-project-id (required) data-api-base-url data-position data-theme data-orb-style data-agent-enabled data-voice-enabled

Programmatic controls

Queue commands any time to control the widget. Use `update` to change options without reloading.

Bello.push(['open']) Bello.push(['close']) Bello.push(['update', {...}])
