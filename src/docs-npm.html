Use the React export for Next.js and React apps. For other setups, import the IIFE bundle and call `Bello.push(['init', ...])`.

Installation checklist

  1. Install the package from npm.
  2. Pick React or the JS bundle integration.
  3. Provide your project ID and API base URL.

Runtime controls

These commands work the same across React and JS.

Bello.push(['open']) Bello.push(['close']) Bello.push(['update', {...}])

Configuration options

`projectId` is required. All other options are optional and can be updated at runtime.

position theme orbStyle widgetTitle widgetButtonTitle agentEnabled voiceEnabled themeVars
